# Виселица (Hangman)

Консольная игра "Виселица" на Java. Игрок должен угадать загаданное слово по одной букве за определённое количество попыток.

## Описание игры

- В начале игры случайным образом выбирается слово из текстового файла со словами.
- Слово изначально скрыто и отображается звёздочками (`*`).
- Игрок вводит по одной русской букве за раз.
- Если буква есть в слове — она открывается на своей позиции.
- Если буквы нет — количество неверных попыток увеличивается, и на экране дорисовывается часть виселицы.
- Всего даётся **7 попыток** на ошибку.
- Игра заканчивается победой (все буквы угаданы) или поражением (закончились попытки).
- После окончания партии можно начать новую игру или выйти.

## Структура проекта

```
├── Main.java              # Точка входа, основной игровой цикл
├── InputChecker.java        # Проверка корректности вводимых букв
├── HangmanPrinter.java # Отрисовка виселицы по этапам (1-7 попыток)
└── GameWords2.txt         # Список слов для игры (по одному слову на строку)
```

## Формат файла со словами

`GameWords2.txt` — обычный текстовый файл, одно слово на строку, кириллицей:

```
программирование
компьютер
клавиатура
университет
```
